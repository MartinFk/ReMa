This ROS package has been modified by Martin Feick. It enables the use of the TRAC-IK algorithm for solving the IK problem. 

1) Install the TRAC-IK library
2) Download this ROS package
3) Copy the package into your Baxter workspace
4) Rebuild your workspace
5) Now, you can use IK-Fast on your Baxter robot
